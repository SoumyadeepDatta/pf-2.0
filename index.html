CSV-jsonify (A CSV to JSON Converter)

Know More  In this project I created a CSV to JSON converter with the help of Java only. I used maven as the build tool. It reads a .csv file and converts it to JSON format and writes in another .txt file. It actually helped some of my friends on a day-to-day basis as they used it (like for serving the data of a dataset using an API). However it is a one-way converter. That means it can't convert JSON back to CSV. I'm planning to add that feature in future.
